# SmartFlow Outreach Command Center

Outbound CRM for Perth real estate agents. Lead tracking, email 
analytics, pace tracker, follow-up cadence management, and full 
sales pipeline with status progression.

Built with Next.js, TypeScript, React, and Tailwind. Supabase for 
persistent data. Deployed on Vercel with n8n workflow automation 
handling the email, SMS, and WhatsApp outreach sequences on the 
backend.

Includes a Demo Mode / Live Mode toggle for safe client 
walkthroughs using sample data.

Live demo: https://realestate-outreach-sand.vercel.app/dashboard
